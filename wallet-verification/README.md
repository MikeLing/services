# Supported platforms:
1. Linux
1. macOS

# Required packages
1. curl
2. gpg

## macOS gpg (2.2.7) installation instructions:
```sh
    $ brew install gnupg 
```

## Ubuntu gpg (1.4.20) installation instructions:
```sh
$ sudo apt-get install gnupg
```

# How to install/run on osx & linux
To run validation process perform following command:
```sh
$ sh ./run.sh
```
